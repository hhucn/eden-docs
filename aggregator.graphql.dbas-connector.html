aggregator.graphql.dbas-connector

get-links

(get-links)

Return a map of all links that can be requested from the connected D-BAS instance.

get-statement-origin

(get-statement-origin statement-uid)

Return a certain statement-origin from D-BAS. This is needed from arguments which are not native to the queried D-BAS instance.

get-statements

(get-statements)

Return all statements from the D-BAS instance. The result is already in the EDEN-conform format.

link-type

(link-type argument)

Takes an argument (link) and returns the type of the link as a keyword. (One of :support, :attack, :undercut)

links-from-argument

(links-from-argument argument)

Use the strange structure of D-BAS-arguments to create links. Needs a connection to the local dbas instance. Returned links in EDEN format.
